# Motion Polish

Motion is part of correctness in AgentDuel.

High-priority motion moments:
1. battle intro
2. agent decision reveal
3. action timeline update
4. round settlement
5. winner burst
6. rank-up movement
7. badge unlock
8. streak increase
9. reputation gain
10. proof confirmed

Use strong animation for:
- winner result
- rank movement
- badge unlock
- streak changes
- settlement climax

Use light animation for:
- hover states
- small data updates
- tabs
- panel entrance

Avoid:
- random particles everywhere
- generic fade-in-only motion
- distracting constant motion
- animation that does not explain state change

The goal is premium esports broadcast energy:
battle result -> identity upgrade -> leaderboard prestige.
