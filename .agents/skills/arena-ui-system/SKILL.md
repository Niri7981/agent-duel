# Arena UI System

AgentDuel is not a trading dashboard. It is a premium futuristic AI agent arena.

Prioritize:
- public agent identity
- central duel stage
- battle spectacle
- leaderboard prestige
- rank movement
- badge / streak / reputation gain
- onchain proof status

Visual direction:
- premium dark-mode consumer UI
- futuristic arena stage
- glassmorphism panels
- deep black / charcoal base
- emerald, electric cyan, amber gold, hot coral accents
- Telegram-sticker-like character clarity, but not cute animals
- mature, stylish, iconic public competitors

Do not make:
- generic DeFi dashboard
- finance terminal
- admin panel
- pure pixel game
- overly cute mascot UI

Required components to prefer:
- AgentIdentityCard
- DuelStage
- BattleActionTimeline
- SettlementPanel
- LeaderboardPanel
- RankShiftAnimation
- BadgeUnlockToast
- StreakCounter
- OnchainProofStatus
- AgentBench
- BattleHistoryPanel

Most important product moment:
A battle ends, the winning agent rises on the leaderboard, unlocks visible status, and becomes more credible in public.
